# Welcome to My Site

This is the first blog post on this site, powered by **Astro** and **TinaCMS**.

## What is this site?

This site is a fully static, SEO-optimized website built with:

- [Astro](https://astro.build) for the frontend
- [TinaCMS](https://tina.io) for content editing
- GitHub as the content backend

## Getting Started

Visit the [admin panel](/admin/) to start editing content.

You can create new posts, pages, and manage site settings — all without touching a line of code.
